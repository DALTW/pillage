# 문서 목적

이 문서는 해커톤 기준 Pencil Village의 MVP 범위를 정의한다. 목표는 10분 안에 처음부터 엔딩까지 플레이 가능한 브라우저 게임을 완성하는 것이다.

# MVP 목표

MVP의 목표는 기능이 많은 게임이 아니라, 완결된 짧은 경험을 만드는 것이다.

핵심 목표:

- 브라우저에서 바로 실행된다.
- 플레이어가 오두막에서 시작해 외부 필드로 나간다.
- 연필 조각을 모아 숲을 확장한다.
- 크레파스를 적용해 세계에 색을 입힌다.
- 문에 도달해 엔딩을 본다.
- 텍스트 없이도 기본 진행을 이해할 수 있다.

# 플레이 시간 목표

MVP 기준 전체 플레이 시간은 약 10분이다.

권장 흐름:

```text
도입 연출: 1분
초기 영역과 1차 확장: 2분
숲 영역 1과 2차 확장: 2분
숲 영역 2와 3차 확장: 2분
숲 영역 3, 문, 엔딩: 2분
여유: 1분
```

# 반드시 구현할 것

## 1. 도입 연출

- 스케치북 화면에서 시작
- 아이의 팔과 연필이 등장
- 오두막, 마을, 숲을 그림
- 연필이 짧아짐
- 스케치북 줌아웃
- 오두막 내부로 전환

## 2. 오두막 내부

- 책상
- 스케치북
- 침대
- 러그
- 모닥불
- 오두막 문
- 스케치북 상호작용
- 외부 필드로 이동

## 3. 외부 필드

- 마을과 숲이 하나의 큰 맵으로 구성
- 플레이어 중심 카메라
- 초기 영역과 숲 영역 1-3
- 해금 전 영역은 이동 불가

## 4. 플레이어 이동

- WASD 또는 방향키 이동
- 충돌 처리
- 상호작용 키 입력

## 5. 상호작용 UX

- 상호작용 대상 반짝임
- 가까이 가면 플레이어 머리 위 키캡 말풍선
- 모든 획득은 상호작용 방식
- 자동 획득 없음

## 6. NPC 4종

- 우체통 주민
- 작은 새
- 다람쥐
- 큰 나무

## 7. 필수 퀘스트 3개

- 편지 찾기
- 나뭇가지 찾기
- 도토리 찾기

각 퀘스트는 연필 조각 1개를 보상으로 준다.

## 8. 큰 나무 크레파스 퀘스트

- 빛나는 잎 찾기
- 크레파스 보상

## 9. 연필 조각 시스템

- 연필 조각 3개로 연필 완성
- 연필 완성 시 오두막 위 느낌표
- 스케치북에서 연필 사용
- 사용 후 연필 UI 초기화

## 10. 숲 확장 3단계

- 숲 영역 1
- 숲 영역 2
- 숲 영역 3

각 확장은 스케치북에서 연필 선이 그어지는 연출로 보여준다.

## 11. 크레파스 시스템

- 크레파스 획득
- 오두막 스케치북에서 적용
- 전체 필드 색상 레이어 활성화

필수 색상:

- 초록색
- 갈색
- 하늘색

## 12. 최종 문과 엔딩

- 3차 확장 후 스케치북에서 문 완성
- 필드 숲 끝에 문 등장
- 문은 오두막 문과 닮은 디자인
- 문을 열면 현실로 돌아가는 엔딩
- 침대에서 눈뜸
- 책상 위 완성된 스케치북 표시

# 선택 구현

시간이 남으면 구현한다.

- 노란색 크레파스
- NPC가 엔딩 스케치북에 작게 등장
- 나뭇잎 흔들림
- 더 부드러운 연필 애니메이션
- 크레파스 색 번짐 효과
- 문 주변 빛 효과
- 간단한 효과음
- 배경음
- 추가 숲 상호작용 오브젝트

# 구현하지 않을 것

MVP에서 제외한다.

- 저장 기능
- 전투
- 체력
- 제한 시간
- 게임오버
- 상점
- 농사
- 마을 건물 추가
- 복잡한 인벤토리
- 랜덤 생성
- 멀티플레이
- 로그인
- 서버
- 랭킹

# 개발 우선순위

## 1순위: 플레이 가능한 핵심 루프

```text
이동 -> 상호작용 -> 연필 조각 -> 스케치북 확장 -> 문
```

## 2순위: 진행 이해 가능성

- 반짝임
- 키캡 말풍선
- 느낌표
- 스케치북 지도

## 3순위: 엔딩까지 완주

- 숲 영역 3까지 확장
- 문 등장
- 엔딩 전환

## 4순위: 아트와 연출 강화

- 색상 적용
- 도입 연출
- 문 열림
- 크레파스 효과

# 개발 마일스톤

## Milestone 1: 기본 게임 프레임

완료 조건:

- Vite + TypeScript 프로젝트 실행
- Canvas 출력
- 게임 루프 작동
- 입력 처리

## Milestone 2: 이동과 씬 전환

완료 조건:

- 플레이어 이동
- 오두막 내부와 외부 전환
- 카메라 추적

## Milestone 3: 상호작용 시스템

완료 조건:

- 반짝이는 상호작용 대상
- 키캡 말풍선
- E 키 상호작용

## Milestone 4: 연필 루프

완료 조건:

- 연필 조각 획득
- 연필 UI 완성
- 오두막 느낌표
- 스케치북에서 숲 확장

## Milestone 5: 퀘스트

완료 조건:

- NPC 말풍선
- 편지, 나뭇가지, 도토리 퀘스트
- 보상 드롭

## Milestone 6: 크레파스

완료 조건:

- 크레파스 획득
- 스케치북에서 색 적용
- 전체 필드 색상 레이어 활성화

## Milestone 7: 엔딩

완료 조건:

- 3차 확장 후 문 등장
- 문 상호작용
- 엔딩 시퀀스
- 최종 스케치북 화면

# 리스크와 대응

## 리스크: 무문자 진행이 이해되지 않음

대응:

- 아이콘을 매우 직관적으로 만든다.
- 퀘스트 오브젝트를 NPC 주변에 배치한다.
- 반짝임과 키캡 표시를 확실히 보이게 한다.

## 리스크: 연출 구현 시간이 과다

대응:

- 연출은 단계별로 최소 구현한다.
- 스케치북 확장은 흐릿한 선이 진해지는 효과만 먼저 구현한다.
- 연필이 직접 움직이는 효과는 후순위로 둔다.

## 리스크: 아트 에셋 제작 지연

대응:

- 초기에는 Canvas 도형과 간단한 선화로 대체한다.
- AI 에셋은 핵심 아이콘부터 생성한다.

## 리스크: 엔딩까지 플레이 시간이 길어짐

대응:

- 필수 퀘스트를 각 단계 1개로 제한한다.
- 연필 조각 필요량은 3개로 유지한다.
- 이동 거리를 짧게 설계한다.

# MVP 완료 판정

아래 조건을 모두 만족하면 MVP 완료로 본다.

- 새로고침 후 처음부터 플레이 가능하다.
- 도입 연출 후 오두막 내부에서 조작 가능하다.
- 외부 필드에서 NPC 퀘스트를 수행할 수 있다.
- 연필 조각 3개로 연필이 완성된다.
- 스케치북에서 3번 확장할 수 있다.
- 크레파스를 적용할 수 있다.
- 최종 문이 등장한다.
- 문을 열면 엔딩이 나온다.
- 일반 대사 없이 진행이 가능하다.
