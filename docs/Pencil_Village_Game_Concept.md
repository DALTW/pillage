# 프로젝트 개요

**Pencil Village**는 스케치북에 그린 작은 중세풍 마을과 숲 속으로 들어가게 된 아이가, 연필 조각을 모아 세계를 확장하고 마지막 문을 열어 현실로 돌아가는 브라우저 힐링 탐험 게임이다.

플레이어는 통나무 오두막에서 시작한다. 아이는 스케치북의 왼쪽 영역에 오두막, 작은 마을, 약간의 숲을 그렸지만 연필이 다 닳아 오른쪽 영역을 완성하지 못했다. 이후 오두막 밖으로 나가자 바깥은 자신이 그린 스케치북 세계가 되어 있다.

게임의 핵심은 **연필로 세계의 선을 그리고, 크레파스로 세계에 색을 입히는 경험**이다.

# 기본 정보

- 임시 제목: **Pencil Village**
- 장르: 힐링, 탐험, 발전
- 플랫폼: 브라우저
- 플레이 방식: 싱글플레이
- 예상 플레이 시간: MVP 기준 약 10분
- 구현 스택: Vite + TypeScript + Canvas API
- 화면 방식: 탑다운 2D, 약간 사선에서 보는 시점
- 저장 기능: MVP에서는 없음

# 한 줄 소개

연필로 그린 스케치북 세계를 탐험하고, 직접 세계를 확장해 원래 현실로 돌아가는 힐링 브라우저 게임.

# 세계관

아이는 통나무 오두막 안에서 혼자 스케치북에 그림을 그리고 있었다. 스케치북의 왼쪽 영역에는 오두막, 작은 마을, 약간의 숲이 그려져 있다. 하지만 연필이 다 닳아 오른쪽 영역은 비어 있다.

아이가 오두막 밖으로 나가자, 바깥세상은 스케치북에 그린 마을과 숲으로 변해 있다. 처음에는 자신이 그린 왼쪽 영역만 존재한다. 오른쪽 영역은 아직 그려지지 않았고, 먼 곳에는 희미한 문만 스케치북 화면에서 보인다.

플레이어는 NPC를 돕고 숲을 탐험해 연필 조각을 모은다. 연필 조각 3개가 모이면 하나의 연필이 완성된다. 완성된 연필은 오두막 안의 스케치북에서 사용되며, 왼쪽 영역에서 오른쪽 영역으로 새로운 숲을 그려 세계를 확장한다.

세계가 충분히 확장되면 숲 끝에 오두막 문과 닮은 문이 나타난다. 플레이어가 그 문을 열면 현실의 침대에서 눈을 뜬다. 책상 위의 스케치북에는 지금까지 탐험하고 색칠한 세계가 완성된 그림으로 남아 있다.

# 핵심 판타지

이 게임의 핵심 판타지는 세 가지다.

## 내가 그린 세계 속으로 들어간다

플레이어는 완성된 판타지 세계에 들어가는 것이 아니라, 방금 전 아이가 직접 그린 스케치북 세계 속으로 들어간다. 그래서 세계는 처음부터 완전하지 않다. 왼쪽 영역만 완성되어 있고 오른쪽 영역은 비어 있다.

## 세계를 직접 확장한다

새로운 지역을 단순히 해금하는 것이 아니라, 스케치북에 직접 그려서 세계를 이어 붙인다. 연필 조각은 경험치가 아니라 **세계를 더 그릴 수 있는 힘**이다.

## 색을 되찾는다

처음 세계는 연필 선화 중심이다. 크레파스를 획득하고 스케치북에 적용하면 전체 필드에 색이 생긴다. 크레파스는 진행 필수 자원이라기보다, 세계를 더 따뜻하게 만드는 특별 보상이다.

# 핵심 재미

- 탐험 중 반짝이는 오브젝트와 NPC를 발견하는 재미
- 텍스트 없이 아이콘을 해석해 퀘스트를 이해하는 재미
- 연필 조각을 모아 연필 UI가 완성되는 성취감
- 오두막으로 돌아가 스케치북에 새 숲을 그리는 확장감
- 크레파스로 흑백 세계에 색을 입히는 보상감
- 마지막 문에 도달해 현실로 돌아가는 감성적인 마무리

# 디자인 기둥

## 1. 텍스트보다 이미지

일반 대사와 설명문은 사용하지 않는다. 진행은 말풍선 아이콘, 반짝이는 오브젝트, 키캡 표시, 스케치북 변화로 전달한다.

## 2. 실패보다 발견

게임오버, 체력, 제한 시간은 없다. 플레이어는 천천히 탐험하고, 필요한 것을 발견하며, 세계를 조금씩 완성한다.

## 3. 성장보다 완성

일반 RPG처럼 강해지는 게임이 아니다. 플레이어의 성취는 캐릭터 능력치가 아니라 **세계가 더 그려지고 색칠되는 변화**로 표현된다.

## 4. 작은 범위의 완결성

MVP는 10분 안에 처음부터 엔딩까지 플레이 가능해야 한다. 숲 영역 확장은 3단계로 제한하고, 각 단계는 짧고 명확한 퀘스트와 수집으로 구성한다.

# 레퍼런스 감각

## 스타듀밸리

- 평화로운 마을 분위기
- 탑다운 탐험
- NPC와의 가벼운 상호작용

## 굶지마

- 손그림 기반의 독특한 아트 감각
- 약간 사선에서 보는 2D 느낌

## 스케치북과 동화책

- 종이 질감
- 연필 선화
- 크레파스로 색이 생기는 따뜻한 변화

# 대상 플레이어

- 짧고 감성적인 브라우저 게임을 좋아하는 플레이어
- 전투보다 탐험과 수집을 선호하는 플레이어
- 텍스트보다 시각적 연출로 진행되는 게임을 좋아하는 플레이어
- 해커톤 시연에서 빠르게 이해할 수 있는 게임을 찾는 심사자

# 최종 경험 목표

플레이어가 엔딩에 도달했을 때 느껴야 하는 감정은 다음과 같다.

```text
내가 처음 본 빈 스케치북이,
탐험과 상호작용을 통해 하나의 완성된 세계가 되었다.
```

# 범위 밖 요소

MVP에서는 다음 요소를 넣지 않는다.

- 전투
- 체력
- 제한 시간
- 상점
- 농사
- 복잡한 인벤토리
- 복잡한 마을 건설
- 온라인 기능
- 저장 기능
- 랜덤 맵 생성

# 프로젝트의 차별점

Pencil Village의 차별점은 **스케치북 UI와 실제 필드가 진행 구조로 연결된다**는 점이다. 플레이어는 단순히 미니맵을 보는 것이 아니라, 스케치북에서 세계를 직접 그리고 색칠한다. 그 결과가 실제 탐험 필드에 반영된다.

이 구조는 해커톤 프로젝트로도 적합하다. 복잡한 전투나 시스템 없이도 명확한 게임 루프, 감성적인 연출, 시각적인 성취감을 만들 수 있기 때문이다.
