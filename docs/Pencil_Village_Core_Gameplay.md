# 문서 목적

이 문서는 Pencil Village의 실제 플레이 규칙을 정의한다. 플레이어가 무엇을 반복하고, 어떤 자원을 얻고, 어떻게 세계를 확장하며, 어떤 조건에서 엔딩에 도달하는지 설명한다.

# 핵심 게임 루프

Pencil Village의 기본 루프는 다음과 같다.

```text
탐험
-> NPC 퀘스트 수행
-> 연필 조각 획득
-> 연필 완성
-> 오두막 복귀
-> 스케치북 상호작용
-> 숲 영역 확장
-> 새 구역 탐험
-> 문 도달
```

이 루프는 총 3회 반복된다. 각 반복마다 숲 영역이 하나씩 확장되고, 마지막에는 문이 실제 필드에 등장한다.

# 전체 진행 흐름

## 1단계: 시작

- 처음 접속 시 스케치북 도입 연출이 나온다.
- 아이가 오두막, 마을, 약간의 숲을 그린다.
- 연필이 다 닳는다.
- 화면 전환 후 아이는 오두막 내부 책상 옆에 서 있다.

## 2단계: 외부 필드 진입

- 플레이어가 오두막 밖으로 나간다.
- 처음에는 오두막과 플레이어만 보인다.
- 짧은 연출로 마을과 숲의 선이 그려지거나 진해진다.
- 초기 마을과 숲 입구가 탐험 가능해진다.

## 3단계: 첫 연필 완성

- 우체통 주민 퀘스트를 수행한다.
- 필드에서 추가 연필 조각을 찾는다.
- 연필 조각 3개가 모이면 연필이 완성된다.
- 오두막 위에 느낌표가 나타난다.

## 4단계: 첫 스케치북 확장

- 플레이어가 오두막으로 돌아간다.
- 스케치북 위에 느낌표가 나타난다.
- 스케치북과 상호작용한다.
- 스케치북 전체가 표시된다.
- 오른쪽 영역 끝의 희미한 문이 보인다.
- 연필로 숲 영역 1이 그려진다.

## 5단계: 반복 확장

- 숲 영역 1과 숲 영역 2에서도 같은 구조가 반복된다.
- 각 확장 단계마다 필수 NPC 퀘스트 1개와 필드 수집이 있다.
- 크레파스는 특별 보상으로 획득할 수 있다.

## 6단계: 문 등장

- 3차 확장 후 스케치북에서 문 완성 연출이 나온다.
- 이후 외부 필드의 숲 끝에 오두막 문과 닮은 문이 등장한다.
- 플레이어가 문과 상호작용하면 엔딩으로 진입한다.

# 연필 조각 시스템

연필 조각은 세계를 더 그릴 수 있게 하는 핵심 자원이다. 일반 RPG의 경험치 역할을 하지만, 표현은 경험치 바가 아니라 연필 조각이 모여 연필이 되는 UI로 한다.

## 기본 규칙

- 연필 조각 3개 = 연필 1개 완성
- 연필 1개 = 숲 영역 1개 확장
- 확장 후 연필 UI는 초기화된다
- 확장은 총 3회 필요하다

```text
1차 연필 완성 -> 숲 영역 1 확장
2차 연필 완성 -> 숲 영역 2 확장
3차 연필 완성 -> 숲 영역 3 확장 및 문 완성
```

## 연필 조각 획득 구조

각 확장 단계마다 다음 구조를 사용한다.

- 필수 NPC 퀘스트 1개 완료 -> 연필 조각 1개
- 필드 상호작용 또는 수집 2개 -> 연필 조각 2개
- 총 3개 획득 시 연필 완성

자동 획득은 없다. 연필 조각은 필드에 떨어진 오브젝트로 보이고, 플레이어가 가까이 가서 상호작용해야 획득된다.

# 연필 UI

연필 UI는 화면 한쪽에 표시한다. 수치 바가 아니라 조각이 모여 하나의 연필이 되는 형태다.

```text
0개: 부러진 짧은 연필
1개: 연필 일부가 붙음
2개: 거의 완성된 연필
3개: 완성된 연필, 반짝임
```

연필이 완성되면 오두막 위에 느낌표가 나타난다. 플레이어가 오두막 근처로 접근하면 아이의 시선 또는 카메라가 오두막 방향을 자연스럽게 유도한다.

# 크레파스 시스템

크레파스는 세계에 색을 입히는 특별 보상이다. 연필이 세계의 선을 확장한다면, 크레파스는 이미 그려진 세계에 색을 더한다.

## 기본 규칙

- 크레파스는 특별 보상으로 등장한다.
- 크레파스를 얻기 위한 상호작용을 완료하면 크레파스가 필드에 떨어진다.
- 플레이어는 떨어진 크레파스와 다시 상호작용해 획득한다.
- 획득 즉시 필드 색이 바뀌지는 않는다.
- 오두막 안에서 스케치북과 상호작용해야 색이 적용된다.
- 색은 전체 필드에 적용된다.
- 적용 순서는 플레이어가 크레파스를 획득한 순서를 따른다.

## 권장 유도 순서

기획상 권장되는 색상 경험 순서는 다음과 같다.

```text
초록색 -> 갈색 -> 하늘색
```

하지만 실제 적용은 플레이어가 찾은 순서대로 한다.

# 스케치북 시스템

스케치북은 게임의 진행 허브다. 오두막 내부 책상 위에 있으며, 언제든 상호작용할 수 있다.

## 상호작용 가능 조건

스케치북은 항상 상호작용 가능하다. 업그레이드 물건이 없으면 현재 스케치북 화면만 보여준다.

## 상태별 처리

- 연필 완성 상태: 다음 숲 영역을 그린다.
- 크레파스 보유 상태: 해당 색을 전체 필드에 적용한다.
- 연필과 크레파스를 둘 다 보유: 연필 적용이 우선된다.
- 아무것도 없음: 현재 스케치북 지도만 보여준다.

## 둘 다 보유했을 때

```text
첫 번째 상호작용: 연필 사용, 새 숲 영역 그리기
두 번째 상호작용: 크레파스 사용, 색상 적용
```

한 번의 상호작용으로 모든 보상을 처리하지 않는다. 각 보상의 연출과 의미를 분리하기 위해 한 번에 하나씩 적용한다.

# 상호작용 방식

모든 핵심 대상은 상호작용 키로 처리한다.

- NPC
- 퀘스트 오브젝트
- 연필 조각
- 크레파스
- 오두막 문
- 스케치북
- 최종 문

자동 획득은 사용하지 않는다.

# 실패 조건

MVP에서는 게임오버가 없다.

- 체력 없음
- 제한 시간 없음
- 전투 없음
- 낙사 없음

진행이 막히는 방식은 실패가 아니라 **아직 필요한 것을 얻지 못한 상태**로 표현한다.

예:

- 연필이 완성되지 않으면 새 숲을 그릴 수 없다.
- 아직 그려지지 않은 숲 영역은 이동할 수 없다.
- 문은 3차 확장 후에만 필드에 등장한다.

# 플레이 템포

MVP 10분 플레이 기준으로 권장하는 템포는 다음과 같다.

```text
도입 연출: 1분 내외
초기 탐험과 1차 확장: 2분
숲 영역 1과 2차 확장: 2분
숲 영역 2와 3차 확장: 2분
문 도달과 엔딩: 1-2분
여유 시간: 1분
```

# 핵심 완성 기준

Core Gameplay 문서 기준으로 MVP가 완성되었다고 판단하려면 다음이 모두 가능해야 한다.

- 플레이어가 오두막 밖으로 나갈 수 있다.
- 필드에서 NPC와 상호작용할 수 있다.
- 연필 조각을 상호작용으로 획득할 수 있다.
- 연필 조각 3개로 연필 UI가 완성된다.
- 오두막으로 돌아가 스케치북에서 숲을 확장할 수 있다.
- 확장을 3회 반복할 수 있다.
- 크레파스를 획득하고 스케치북에서 색을 적용할 수 있다.
- 최종 문이 필드에 등장한다.
- 문과 상호작용하면 엔딩이 나온다.
