## 현재 목표

`Pencil Village`를 Vite + TypeScript + Canvas API 기반의 10분 MVP 브라우저 게임으로 구현한다.

완료 기준:

- 새로고침 후 처음부터 플레이 가능하다.
- 도입 연출, 오두막, 외부 필드, 스케치북 확장, 크레파스 적용, 최종 문, 엔딩이 연결된다.
- 일반 대사 없이 아이콘과 시각 힌트로 기본 진행이 가능하다.
- MVP 제외 범위는 추가하지 않는다.

- [ ] 무문자 진행이 이해되지 않을 수 있음
- [ ] 연출 구현 시간이 과다할 수 있음

대응 원칙:

- 아이콘은 구체적인 물건 형태로 만든다.
- 연출은 먼저 최소 구현하고, 세부 애니메이션은 후순위로 둔다.
- 초기 아트는 Canvas 선화와 단순 도형으로 대체 가능하게 만든다.
- 퀘스트 오브젝트는 NPC와 너무 멀리 배치하지 않는다.
