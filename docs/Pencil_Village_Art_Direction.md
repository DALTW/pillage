# 문서 목적

이 문서는 Pencil Village의 시각 방향, 스케치북 질감, 연필 선화, 색상 레이어, 에셋 제작 기준을 정의한다. 구현은 Canvas 기반이지만 일부 아이콘과 오브젝트는 AI 생성 SVG/PNG 에셋을 사용할 수 있다.

# 전체 아트 방향

Pencil Village의 기본 시각 컨셉은 **연필로 그린 스케치북 세계**다.

핵심 키워드:

- 스케치북
- 종이 질감
- 연필 선화
- 중세풍 작은 마을
- 통나무 오두막
- 힐링 숲
- 크레파스 색상
- 동화책 감성

# 화면 시점

- 탑다운 2D
- 약간 사선에서 보는 느낌
- 캐릭터와 오브젝트의 앞면이 일부 보임
- 스타듀밸리처럼 이동과 탐험이 직관적이어야 함
- 굶지마처럼 손그림 느낌 허용

# 기본 색감

초기 상태는 무채색에 가깝다.

- 종이 배경: 따뜻한 아이보리 또는 연한 베이지
- 선: 연필 흑연색, 너무 완전한 검정은 피함
- 그림자: 옅은 회색
- 인터랙션 반짝임: 부드러운 흰색 또는 연한 노란빛

크레파스 적용 전에는 전체적으로 차분한 선화 느낌을 유지한다.

# 스케치북 질감

종이 질감은 게임의 핵심 분위기다.

구현 방식:

- Canvas 배경에 연한 노이즈 또는 종이 텍스처 패턴 사용
- 가장자리에는 살짝 어두운 종이 그림자
- 스케치북 UI에서는 종이의 테두리를 더 명확히 표현

주의:

- 질감이 너무 강하면 캐릭터와 오브젝트가 잘 보이지 않는다.
- MVP에서는 단순한 아이보리 배경과 얇은 노이즈만으로 충분하다.

# 연필 선화 스타일

오브젝트는 완벽한 벡터 선보다 약간 흔들리는 선이 좋다.

선화 규칙:

- 선 두께는 1-3px 범위
- 같은 오브젝트 안에서도 선 두께가 조금 달라질 수 있음
- 모서리는 약간 둥글게
- 완전히 정교한 도면보다 손그림 느낌 우선

# 캐릭터

플레이어는 스케치북 속에 들어온 아이로 표현한다.

권장 형태:

- 작고 단순한 실루엣
- 머리와 몸이 명확히 구분됨
- 이동 방향에 따라 간단한 4방향 또는 8방향 스프라이트
- 초기에는 무채색 선화 중심

MVP에서는 애니메이션을 단순화할 수 있다.

- 정지 프레임
- 걷기 2프레임
- 방향 전환만 구현

# 오두막

오두막은 게임의 시작과 끝을 연결하는 핵심 오브젝트다.

특징:

- 통나무집
- 작은 굴뚝 또는 모닥불 느낌
- 따뜻하고 안전한 분위기
- 최종 문과 디자인적으로 연결됨

오두막 문은 최종 문의 디자인 기준이 된다.

# 마을

마을은 중세풍이지만 복잡하지 않아야 한다.

권장 요소:

- 작은 집 몇 채
- 우체통
- 낮은 울타리
- 돌길 또는 흙길
- 작은 광장

MVP에서는 마을 건물 추가나 발전은 구현하지 않는다. 마을은 시작 구역과 첫 퀘스트 공간의 역할을 한다.

# 숲

숲은 확장되는 주요 공간이다.

공통 요소:

- 나무
- 덤불
- 풀
- 그루터기
- 작은 공터
- 길

구역별 차이:

- 초기 숲: 단순한 숲 입구
- 숲 영역 1: 새와 나뭇가지가 어울리는 숲길
- 숲 영역 2: 다람쥐와 도토리가 어울리는 공터
- 숲 영역 3: 큰 나무와 문이 있는 상징적 공간

# 크레파스 색상 레이어

색상은 오브젝트 자체를 완전히 다시 그리는 방식보다, 레이어를 켜는 방식이 적합하다.

## 초록색

적용 대상:

- 풀
- 나뭇잎
- 덤불

의도:

- 숲이 살아나는 첫 느낌

## 갈색

적용 대상:

- 나무줄기
- 오두막 일부
- 울타리
- 흙길

의도:

- 세계가 더 따뜻하고 안정적으로 보임

## 하늘색

적용 대상:

- 물
- 하늘빛 효과
- 문 주변 빛

의도:

- 마지막 공간과 현실 복귀의 느낌 강화

## 노란색

MVP에서는 선택 보상으로만 고려한다.

적용 대상:

- 꽃
- 창문 불빛
- 반짝임

# UI 아트

## 연필 UI

연필 조각이 하나씩 붙어 연필이 완성되는 형태다.

상태:

```text
0/3: 부러진 연필
1/3: 첫 조각 추가
2/3: 거의 완성
3/3: 완성, 반짝임
```

## 크레파스 UI

획득한 크레파스는 작고 단순한 아이콘으로 표시한다. 색 적용 전과 적용 후를 구분할 필요가 있다면, 적용 전은 살짝 흐리게 표시하고 적용 후는 진하게 표시한다.

## 키캡 아이콘

상호작용 키를 표시하는 작은 키캡 이미지다. 말풍선 안에 표시된다.

권장:

- 키캡 모양
- 내부에 `E` 표시 가능
- 손 아이콘을 함께 넣을 수 있음

# AI 생성 에셋 가이드

AI로 생성할 수 있는 에셋:

- NPC 아이콘
- 퀘스트 아이콘
- 키캡 UI
- 연필 조각
- 크레파스
- 오두막 선화
- 나무 선화
- 문 선화

에셋 제작 기준:

- 투명 배경 PNG 또는 SVG 권장
- 선화 중심
- 너무 많은 색 사용 금지
- 같은 두께와 분위기 유지
- 작은 크기에서도 알아볼 수 있어야 함

# 에셋 프롬프트 예시

simple hand-drawn pencil fragment icon,

transparent background,

readable at small size
```

## 크레파스

```

```text

hand-drawn pencil line art,

```

# 애니메이션 방향

MVP에서 필요한 애니메이션:

- 오브젝트 반짝임
- 플레이어 걷기
- 연필 UI 조각 추가
- 스케치북 확장 시 선이 그려지는 효과
- 크레파스 색 적용 효과
- 문 열림 효과

선택 애니메이션:

- 종이 질감 미세 흔들림
- 나뭇잎 흔들림
- NPC 반응
- 엔딩 빛 효과

# 아트 우선순위

MVP에서 우선 제작할 것:

```text
1. 플레이어 캐릭터
2. 오두막 내부 주요 오브젝트
3. 오두막 외부
4. 마을 기본 오브젝트
5. 숲 기본 오브젝트
6. NPC 4종
7. 퀘스트 아이콘 4종
8. 연필 조각과 연필 UI
9. 크레파스 아이콘
10. 최종 문
```

# 품질 기준

아트가 완성되었다고 판단하려면 다음 기준을 충족해야 한다.

- 처음 봐도 스케치북 세계라는 느낌이 든다.
- 오브젝트가 작은 크기에서도 알아보기 쉽다.
- 반짝임과 키캡 UI가 명확하다.
- 크레파스 적용 전후 차이가 확실하다.
- 최종 스케치북 그림이 엔딩 보상처럼 느껴진다.
