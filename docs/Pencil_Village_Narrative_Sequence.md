# 문서 목적

이 문서는 Pencil Village의 도입, 외부 전환, 스케치북 확인, 확장, 색칠, 엔딩 연출을 장면 단위로 정의한다. 게임 안에서 텍스트 없이 세계관과 목표를 전달하기 위한 연출 기준이다.

# 전체 연출 구조

```text
도입: 아이가 스케치북 왼쪽 영역을 그림
시작: 오두막 내부에서 조작 시작
전환: 밖으로 나가면 스케치북 세계가 펼쳐짐
반복: 연필 완성 후 스케치북에서 숲 확장
보상: 크레파스 적용으로 세계에 색이 생김
결말: 숲 끝의 문을 열고 현실로 돌아감
```

# 도입 연출

도입은 게임의 정체성을 전달하는 가장 중요한 시퀀스다. 텍스트 없이 다음 정보를 전달해야 한다.

- 아이가 스케치북에 세계를 그렸다.
- 왼쪽 영역만 그려져 있다.
- 연필이 다 닳아 더 그릴 수 없다.
- 이후 아이가 그 세계 속에서 움직이게 된다.

## 장면 1: 스케치북 클로즈업

처음 접속하면 스케치북 화면이 보인다.

화면 구성:

- 화면 왼쪽에는 짙은 색의 나무 테이블이 보인다.
- 화면 오른쪽은 스케치북으로 이어진다.
- 카메라는 스케치북의 왼쪽 영역을 보고 있다.

의도:

- 플레이어가 현실의 책상 위 스케치북을 보고 있음을 이해하게 한다.
- 아직 오른쪽 영역 전체는 보이지 않는다.

## 장면 2: 팔과 연필

화면 아래에서 아이의 팔이 올라온다. 손에는 연필이 들려 있다.

행동:

- 아이가 스케치북 왼쪽 영역 중앙에 통나무 오두막을 그린다.
- 이후 주변에 작은 중세풍 마을과 약간의 숲을 그린다.

표현:

- 연필 선이 실제로 그어지는 느낌을 준다.
- 완성된 선은 진한 흑연 선화로 남는다.

## 장면 3: 연필이 짧아짐

그림을 그릴수록 손에 든 연필이 점점 짧아진다. 마지막 선을 긋고 나면 연필은 거의 다 닳아 있다.

의도:

- 왜 오른쪽 영역이 비어 있는지 설명한다.
- 연필 조각 수집의 동기를 만든다.

## 장면 4: 스케치북 줌아웃

카메라가 줌아웃되며 스케치북 전체가 화면에 들어온다.

표시 내용:

- 왼쪽 영역: 오두막, 마을, 숲 일부가 그려져 있다.
- 오른쪽 영역: 비어 있다.

문은 아직 이 장면에서 명확히 보이지 않아도 된다. 문은 이후 첫 스케치북 확인에서 목표로 드러난다.

## 장면 5: 전환

화면이 잠시 어두워졌다가 다시 밝아진다. 아이는 오두막 내부의 책상 옆에 서 있다.

이 시점부터 플레이어 조작이 시작된다.

# 오두막 내부 시작

플레이어는 통나무 오두막 내부에서 시작한다.

오브젝트:

- 책상
- 스케치북
- 침대
- 러그
- 모닥불
- 오두막 문

스케치북은 항상 상호작용 가능하다. 처음 상호작용하면 현재 그려진 왼쪽 영역과 빈 오른쪽 영역을 볼 수 있다. 업그레이드 물건이 없으면 단순히 스케치북 화면만 보여준다.

# 첫 외부 전환 연출

플레이어가 오두막 문을 통해 밖으로 나가면 짧은 외부 전환 연출이 나온다.

권장 길이:

```text
3-5초
```

## 장면 구성

```text
1. 화면이 밝아진다.
2. 처음에는 플레이어와 오두막만 보인다.
3. 배경은 종이 질감의 스케치북처럼 보인다.
4. 주변에 흐릿한 마을과 숲의 밑그림이 나타난다.
5. 선이 점점 진해지거나 새로 그려진다.
6. 초기 마을과 숲 입구가 활성화된다.
```

의도:

- 현실의 오두막 밖이 스케치북 세계로 변했음을 전달한다.
- 플레이어가 지금부터 탐험할 초기 영역을 인식하게 한다.

# 첫 스케치북 확인 연출

플레이어가 첫 연필을 완성하고 오두막으로 돌아와 스케치북과 상호작용하면, 첫 스케치북 확인 연출이 나온다.

## 장면 구성

```text
1. 전체 화면 스케치북 UI가 열린다.
2. 왼쪽 영역의 오두막, 마을, 숲이 보인다.
3. 카메라가 왼쪽 영역에서 오른쪽 영역으로 천천히 이동한다.
4. 오른쪽 영역 끝에 희미한 문이 보인다.
5. 문이 살짝 반짝인다.
6. 카메라가 현재 확장 가능한 위치로 돌아온다.
7. 연필 선이 그어지며 숲 영역 1이 완성된다.
```

의도:

- 최종 목표가 문이라는 것을 텍스트 없이 보여준다.
- 현재 목표는 문에 도달할 만큼 세계를 확장하는 것임을 전달한다.

# 숲 확장 연출

숲 확장은 총 3회 발생한다.

공통 연출:

```text
1. 오른쪽 영역에 흐릿한 밑그림이 보인다.
2. 연필 선이 그어지는 효과가 나온다.
3. 흐릿한 선이 진한 선으로 바뀐다.
4. 새 숲 영역이 스케치북에 추가된다.
5. 필드로 돌아가면 해당 영역이 실제로 이동 가능해진다.
```

## 1차 확장

- 숲 영역 1이 생긴다.
- 문은 아직 멀리 희미하게 남아 있다.

## 2차 확장

- 숲 영역 2가 생긴다.
- 문에 가까워졌다는 느낌을 스케치북 배치로 보여준다.

## 3차 확장

- 숲 영역 3이 생긴다.
- 문까지 길이 이어진다.
- 문이 더 선명해지고 완성된다.

# 크레파스 색칠 연출

크레파스를 획득한 뒤 오두막에서 스케치북과 상호작용하면 색칠 연출이 나온다.

규칙:

- 연필과 크레파스를 둘 다 가지고 있으면 연필 적용이 우선된다.
- 크레파스는 재상호작용으로 적용된다.
- 색은 전체 필드에 적용된다.
- 색 적용 순서는 플레이어가 크레파스를 얻은 순서다.

## 연출 방식

```text
1. 스케치북 UI가 열린다.
2. 해당 크레파스 색이 화면 한쪽에서 나타난다.
3. 색이 스케치북 위 특정 요소에 번지듯 적용된다.
4. 필드로 돌아가면 같은 색상 레이어가 활성화되어 있다.
```

예시:

- 초록색: 풀, 나뭇잎, 덤불
- 갈색: 나무줄기, 오두막, 울타리, 흙길
- 하늘색: 물, 하늘빛 효과, 문 주변 빛

# 문 완성 연출

3차 확장 후 스케치북에서 문 완성 연출이 나온다.

장면 구성:

```text
1. 스케치북 오른쪽 끝의 문 밑그림이 보인다.
2. 연필 선이 문틀과 손잡이를 진하게 그린다.
3. 문이 오두막 문과 닮은 형태로 완성된다.
4. 문 주변에 약한 빛이 생긴다.
5. 필드로 돌아가면 숲 끝에 같은 문이 등장한다.
```

# 최종 문 상호작용

필드의 숲 끝에 오두막 문과 닮은 문이 서 있다.

상호작용 흐름:

```text
1. 문이 은은하게 반짝인다.
2. 플레이어가 가까이 가면 머리 위에 키캡 말풍선이 뜬다.
3. 상호작용 키를 누른다.
4. 문이 열린다.
5. 문 안에서 하얀 빛이 나온다.
6. 화면이 밝게 전환된다.
```

# 엔딩 연출

## 장면 1: 침대에서 눈뜸

아이는 현실의 오두막 또는 방 안 침대에서 눈을 뜬다. 게임 시작 때의 오두막 내부와 비슷한 공간이어도 된다.

의도:

- 아이가 현실로 돌아왔음을 전달한다.
- 스케치북 세계가 꿈처럼 보이지만 완전히 사라지지는 않았음을 암시한다.

## 장면 2: 책상 위 스케치북

카메라가 책상 위 스케치북으로 이동한다.

스케치북에는 다음이 보인다.

- 왼쪽 영역부터 오른쪽 영역까지 확장된 전체 맵
- 플레이어가 적용한 크레파스 색상
- 오른쪽 끝의 열린 문

NPC 작은 그림은 MVP에서는 제외한다. 시간이 남으면 작은 새, 다람쥐, 큰 나무를 작게 넣어도 된다.

## 장면 3: 종료

마지막 화면은 완성된 스케치북 그림으로 유지한다.

가능한 표현:

```text
스케치북 전체가 화면에 남고,
열린 문 주변의 빛이 천천히 사라진다.
```

텍스트 엔딩 문구는 사용하지 않아도 된다.

# 연출 우선순위

MVP에서 반드시 필요한 연출:

- 도입 스케치북 그리기
- 첫 외부 전환
- 스케치북 확장 3회
- 문 완성
- 엔딩 스케치북

여유가 있으면 추가할 연출:

- 손의 연필 움직임 세부 애니메이션
- 크레파스 색 번짐 애니메이션
- NPC 보상 드롭 애니메이션
- 문 열림 효과
- 엔딩에서 작은 NPC 그림
